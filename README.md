
# Survey Data Cleaning & Management Guidelines

This repository contains a step-by-step guide on how to fix and organize survey data. Think of raw data like fresh vegetables from a farm—before you can cook them, you need to wash them and remove the bad pieces. Data cleaning does the same thing by making sure your information is accurate and ready for analysis.

**Why This Guide is Helpful:**
* **Fixes Mistakes:** It helps you find wrong numbers, missing answers, and repeated records that could lead to the wrong conclusions.
* **Clear Rules:** It provides standard codes to use when an answer is missing or refused, so the dataset stays organized.
* **Team Roles:** It explains who does what, from the Data Entry Clerk to the Principal Investigator.
* **Step-by-Step Process:** It includes a 12-step journey, starting with backing up your files and ending with locking the final clean dataset.

**Key Rules to Remember:**
* **Never Edit the Original:** Always make a copy of the raw data before you start.
* **Log Every Change:** Every single correction must be written down in a "Data Cleaning Log".
* **Keep Data Private:** Personal information like names and phone numbers must be hidden to protect people's privacy.
* **No Guessing:** If an answer is confusing, flag it and ask a manager instead of guessing.

**Using this Guide:**
You can follow these instructions whether you are using **Excel**, **STATA**, **R**, or **Python**. By following these steps, you make sure your research is something people can trust.
